    Java SE 21程式語言設計¶

    班次: 2025/11/24 網站: https://www.uuu.com.tw/Course/Show/2131/Java-SE-21%E7%A8%8B%E5%BC%8F%E8%AA%9E%E8%A8%80%E8%A8%AD%E8%A8%88

    課程大綱¶

    1. Java技術概論
      • 了解 Java 平台的運作原理 (JVM, JRE, JDK)
      • 認識 Java SE 版本演進與特色
      • 撰寫並執行第一個 Java 程式
    2. 基本資料型態、運算子與流程控制敘述
      • 掌握八大基本資料型態 (Primitive Types)
      • 熟悉變數宣告與範疇 (Scope)
      • 運用算術、比較、邏輯與位元運算子
      • 使用 if-else, switch, for, while 等流程控制
    3. 文字、日期、時間與數字物件
      • 使用 String, StringBuilder 處理文字
      • 運用 Java Date/Time API (LocalDate, LocalDateTime, ZonedDateTime)
      • 數字格式化與 BigDecimal 精確運算
    4. 類別與物件
      • 理解物件導向核心概念 (Class vs Object)
      • 定義類別、屬性 (Field) 與方法 (Method)
      • 使用建構子 (Constructor) 初始化物件
      • 關鍵字 this 的使用
    5. 設計類別與列舉(enum)
      • 封裝 (Encapsulation) 與存取修飾子 (Access Modifiers)
      • 靜態成員 (static) 與實例成員的區別
      • 定義與使用 Enum 列舉型態
    6. 實作繼承與使用紀錄
      • 繼承 (Inheritance) 的語法與應用 (extends)
      • 方法覆寫 (Overriding) 與 super 關鍵字
      • 使用 Java 14+ 的 Record 類別簡化資料載體
    7. 介面與泛型(Generics)
      • 定義與實作介面 (Interface)
      • 介面的預設方法 (Default Methods) 與靜態方法
      • 泛型類別與泛型方法的基礎應用
    8. 陣列與迴圈
      • 一維與多維陣列的宣告與操作
      • 使用 Enhanced for-loop 走訪陣列
      • 陣列複製與 Arrays 工具類別
    9. 集合(Collections)
      • List, Set, Map 介面的特性與選擇
      • ArrayList, HashSet, HashMap 的實作與應用
      • 集合的走訪與排序 (Comparable/Comparator)
    10. 巢狀類別與Lambda運算式
      • 內部類別 (Inner Class) 與匿名內部類別
      • Lambda 表達式的語法與應用場景
      • 方法參考 (Method Reference)
    11. Java 8串流API
      • Stream API 的建立與操作 (Intermediate vs Terminal operations)
      • 常見操作：filter, map, reduce, collect
      • 平行串流 (Parallel Stream) 的概念
    12. 例外控制、記錄(Records)與除錯
      • try-catch-finally 區塊的使用
      • Checked vs Unchecked Exception
      • 自定義例外與 throw/throws
      • 使用斷言 (Assertion) 與除錯技巧
    13. 標記(Annotations)
      • 內建註解 (@Override, @Deprecated, @SuppressWarnings)
      • 自定義註解的定義與使用
      • 了解註解在框架中的應用原理
    14. 開發資料庫應用程式
      • JDBC 核心 API 介紹 (Connection, Statement, ResultSet)
      • 執行 SQL 查詢與更新
      • PreparedStatement 的使用與 SQL Injection 防護
    15. 安全性
      • Java 安全模型概論
      • 基礎加密與解密概念
      • 安全編碼實務 (Secure Coding Practices)
    16. 進階泛型(Generics)
      • 泛型通配符 (Wildcards: ?, ? extends T, ? super T)
      • 泛型邊界 (Bounded Type Parameters)
      • 泛型擦除 (Type Erasure) 的影響
    17. 設計模式、常式運算式與IO watch服務
      • 常見設計模式 (Singleton, Factory, Observer 等)
      • 正規表示式 (Regular Expressions) 的語法與 Pattern/Matcher
      • NIO.2 Path 與 WatchService 監控檔案系統變更

    學習重點與目標¶

    課程目標¶

    本課程使用最新的Java SE 21實作Java SE應用程式，課程內容涵蓋物件導向程式設計、Java程式設計語言的語法與程式設計技巧、使用Java程式語言結構與運算子、使用Java核心API，例如集合、串流、輸出入與執行緒、佈署Java SE應用程式。

    學會技能¶

    1. 學會Java程式語言基本語法
    2. 學會Java應用程式開發
    3. 學會物件導向原理如何應用到Java程式開發
    4. 認識Java程式的資料結構
    5. 透過Java程式存取檔案資料
    6. 透過Java程式存取資料庫資料
    7. 使用Java語言撰寫多執行緒應用程式
    8. 使用Lambda運算式處理集合物件
    9. 開發具備安全性的Java程式
